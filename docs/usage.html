skypebot

A module made for coding text command bots for Skype, with Python.



Explanation of the Quickstart

Here's the code from the quickstart:

import skypebot

 email = input('Email: ')
 password = input('Password: ')
 message = skypebot.Message('!') #initialise message with the prefix '!'
 
 #example of a message handler
 @message.handler() #custom handler
 def messagehandle(message):
   print(message.user.name+': '+message.content) #print the message's sender's name and message content

 #example of a basic command
 @message.command('hi') #hi command - can be accessed in Skype by typing '!hi'
 def hi(message, arguments):
   message.chat.sendMsg('hello') #send 'hello' back.
 
 #example of a command with arguments
 @message.command('repeat', ['times', 'thing']) #repeat command - can be accessed in Skype by typing '!repeat  '
 def repeater(message, arguments):
   string = arguments.thing + '\n' #create a string with the thing to repeat
   string = string * int(arguments.times) #repeat that thing multiple times
   message.chat.sendMsg(string) #send to chat

 skypebot.login(message, email, password) #log in to the bot

To initialise the command/message handler, you have to do message = skypebot.Message('your-prefix'). Note that '/' is not a valid prefix, as Skype already uses that for other in-built commands. It will raise a ValueError.

Commands can be registered with @message.command(name='command name'), and message handlers with @message.handler().

For commands, you have to have 2 parameters in your function: message, and arguments. If you don't use arguments in your command, you can safely ignore that parameter (but still include it), but if you do use arguments, specify them in the decorator above it by adding the parameter arguments=['list', 'of', 'arguments'].

For handlers, you only need to have 1 parameter, that is, message.

The arguments parameter returns a ParsedArgumentsObject. For example, to get a specified argument 'word' (which would be specified in the decorator, as mentioned above), inside the function you would use arguments.word.

The message parameter returns a skpy.message.SkypeMsg object. I highly suggest you check out SkPy's documentation and API reference here to use it in your code.

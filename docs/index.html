skypebot v0.1.0

A module made for coding text command bots for Skype, with Python.



About

skypebot is a module made for coding text command bots for Skype, with Python.
Currently it's still in development stages so there isn't support for much yet.
In the future, I plan to add support for more things (quotes, custom interactive polls, automatic help command, other chat events, etc.) but currently there is support for:

  • Text commands (e.g. !help, !say, etc.)
  • Message handling

Requirements

  • A Skype account to use for the bot (this is just a normal Skype account)
  • Sufficient knowledge of Python

Installing

Via pip (in code source directory):

$ pip install git+https://github.com/InterCozmic/skypebot

Via git (in code source directory):

$ git clone https://github.com/InterCozmic/skypebot

Quickstart

Navigate to the module (usually located in <source-directory>/skypebot) in command line and run:

$ python __init__.py

It will prompt you for the email and password of the bot account, and then it should run! The account provided should now have two preset commands:

  • !repeat [number of times] [thing to repeat] - returns the thing you want to repeat, the specified number of times, in a message (i.e. typing '!repeat 3 hello' in a group chat with the bot in it will reply:
    hello
    hello
    hello
  • !hi - responds with 'Hello!'
